===========================================
          PakGamer Website Project
===========================================

Project Type:
-------------
Local, small-scale gaming website built using:
- HTML5
- CSS3
- JavaScript (basic interactivity)
(No backend / No database used)

-------------------------------------------
Website Overview:
-------------------------------------------
PakGamer is a local gaming website that showcases:
- Home Page with Hero, About, Games, Experience, and Contact sections
- Dedicated pages for About, Games, Experience, and Contact
- Responsive and modern dark-neon design
- Simple scroll animations and smooth navigation
- No backend — works completely offline

-------------------------------------------
How to Run the Website:
-------------------------------------------
1. Place all files in the same folder structure shown above.
2. Double-click on "index.html" to open it in your browser.
3. You can navigate between pages using the navbar.
4. The same CSS and JS are shared across all pages.
5. Works offline — no server required.

-------------------------------------------
How We Built It:
-------------------------------------------
1. Created folder structure: css/, js/, images/, and main HTML files.
2. Designed layout using HTML5 and CSS3 (with flexbox for responsiveness).
3. Added hero, about, games, experience, and contact sections.
4. Added simple JavaScript for scroll animation and smooth scrolling.
5. Used consistent color theme: black background with neon green highlights.
6. Linked all pages to use the same stylesheet and JS file.

-------------------------------------------
Customization:
-------------------------------------------
- Replace images in the /images folder with your own.
- Edit text in HTML files to personalize.
- Modify colors or effects in css/style.css if needed.
- Add more game cards or experience stories as desired.
